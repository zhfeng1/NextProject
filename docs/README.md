## 📊 项目概览

### 技术栈对比

| 组件 | v1.0（当前） | v2.0（规划） |
|------|-------------|-------------|
| **后端** | Python + FastAPI + SQLite | Python + FastAPI + PostgreSQL + Celery |
| **前端** | Vue 3 (CDN) | Vue 3 + TypeScript + Vite + Pinia |
| **数据库** | SQLite | PostgreSQL 15 + Redis 7 |
| **任务队列** | 内存队列 | Celery + Redis |
| **对象存储** | 本地磁盘 | MinIO |
| **部署方式** | Docker 进程 | Kubernetes (Apollo) |
| **监控** | 无 | Prometheus + Grafana |

### 工期估算

| 阶段 | 后端 | 前端 | 总计 |
|------|------|------|------|
| **Phase 1** | 1 个月 | 3 周 | 7 周 |
| **Phase 2** | 2 个月 | 5 周 | 13 周 |
| **Phase 3** | 持续优化 | 3 周 | 持续 + 3 周 |
| **合计** | ~3 个月 | ~2.5 个月 | **约 3.5 个月** |

## 🔧 开发环境配置

### 最低要求

- Docker 20.10+
- Docker Compose 2.0+
- Python 3.12+（如需本地开发）
- Node.js 20+（前端开发）

### 推荐配置

- **CPU**：8 核以上
- **内存**：16 GB 以上
- **磁盘**：50 GB 可用空间（SSD）
- **网络**：访问 Harbor (192.168.1.18) 和 Apollo (192.168.1.15:90)
